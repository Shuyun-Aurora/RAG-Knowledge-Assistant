## 2. 基础测试数据约定

为减少测试用例重复描述，后续自动化实现阶段统一使用如下基础测试数据或等价 fixture：

| 标识 | 说明 |
| --- | --- |
| `teacher_user` | 教师用户，`role='teacher'` |
| `student_user` | 学生用户，`role='student'` |
| `course_1` | 有效课程记录，供练习和帖子挂载 |
| `exercise_set_1` | 课程 `course_1` 下的习题集，包含单选、多选、填空三类题目 |
| `exercise_single_1` | `exercise_set_1` 中的单选题 |
| `exercise_multi_1` | `exercise_set_1` 中的多选题 |
| `exercise_blank_1` | `exercise_set_1` 中的填空题 |
| `post_real_1` | 实名帖子，作者为 `student_user` |
| `post_anon_1` | 匿名帖子，作者为 `teacher_user` |
| `comment_parent_1` | `post_real_1` 下的一级评论 |
| `comment_reply_1` | 回复 `comment_parent_1` 的二级评论 |
